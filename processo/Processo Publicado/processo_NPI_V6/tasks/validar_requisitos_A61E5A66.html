Tarefa: Validar Requisitos
Esta tarefa consite na validação dos requisitos pelo cliente.
Disciplinas: Requisitos
Relacionamentos
FunçõesExecutor Primário:
  • Equipe de desenvolvimento
  • Líder Técnico
Executores Adicionais:
EntradasObrigatório:
  • Documento de Visão
    		Opcional:
    • Nenhum
    Saídas
    • Documento de Especificação dos Requisitos
      Etapas
      1) Validar requisitos pelo cliente

      Os requisitos identificados, o qual estão contidos no documento de visão, devem ser validados pelo cliente.

      O cliente tem o poder de adicionar, modificar e excluir requisitos.

      2) Realizar alterações nos requisitos
      Caso seja realizado quaisquer alterações nos requisitos, estas devem ser registradas no documento de visão.
