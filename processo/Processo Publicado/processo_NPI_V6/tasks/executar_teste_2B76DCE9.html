Tarefa: Executar Teste
Esta tarefa consiste na execução dos testes no produto.
Disciplinas: Testes
Relacionamentos
FunçõesExecutor Primário:
  • Testador
Executores Adicionais:
EntradasObrigatório:
  • Documento de Especificação de Caso de Uso
  • Planilha de Teste
  • Release do Produto
    		Opcional:
    • Nenhum
    Saídas
    • Planilha de Teste
      Etapas
      1) Executar os testes

      Os testes devem ser executados de acordo com os cenários descritos no documento de especificação de caso de uso. Todos os testes realizados devem ser inseridos na planilha de teste. Cada execução dos testes deve gerar uma nova aba na planilha.
