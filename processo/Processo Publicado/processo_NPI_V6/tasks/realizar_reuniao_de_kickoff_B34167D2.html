Tarefa: Realizar Reunião de Kickoff
Esta tarefa tem como objetivo realizar uma reunião de kickoff com o intuito de apresentar os projetos aos integrantes do NPI, bem como alocar os recursos necessários para cada projeto e escolher os líderes técnicos.
Disciplinas: Iniciar Projeto
Relacionamentos
FunçõesExecutor Primário:
  • Equipe de desenvolvimento
  • Professor Supervisor
Executores Adicionais:
EntradasObrigatório:
  • Documento de Visão
    		Opcional:
    • Nenhum
    Saídas
    • Ata de Reunião
      Etapas
      1) Realizar Reunião de Kickoff

      A reunião de kickoff tem como intuito apresentar os projetos a todos os envolvidos do NPI. O responsável por essa reunião e o seu conteúdo é o professor supervisor. Nessa reunião devem ser escolhidos os líderes técnicos e alocados os recursos que irão trabalhar em cada projeto e determinado quais suas responsabilidades. Nesta atividade todos os participantes terão que se comprometer com o projeto.

      Deve ser criada uma ata de reunião documentando tudo o que foi dito durante a Reunião de Kickoff.
