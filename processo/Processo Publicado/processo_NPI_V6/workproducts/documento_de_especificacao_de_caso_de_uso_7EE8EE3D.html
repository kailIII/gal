Artefato: Documento de Especificação de Caso de Uso
Cada caso de uso especificados na Sprint, devidamente documentados no formato dos templates de Caso de Uso CRUD ou Relatório Específico, deve gerar um documento de caso de uso específico. O nome documento de especificação de caso de uso é simbólico. Na verdade, o conjunto de todos os casos de uso específicos forma a especificação de casos de uso. Porém, não irá existir um documento q reunirá todos os casos de uso e sim, cada caso de uso deve gerar um documento.
